# Onboardly

A simple React component library for Onboardly.

## Installation

```bash
npm install onboardly
# or
yarn add onboardly
# or
pnpm add onboardly
```

## Usage

```jsx
import { Onboardly } from 'onboardly';

function App() {
  return <Onboardly />;
}
```
